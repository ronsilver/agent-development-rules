# Workflow: Revisar PR

## Pasos

1. **Obtener info del PR**
   ```bash
   gh pr view
   gh pr diff
   ```

2. **Revisar cambios**
   - Verificar código
   - Verificar tests
   - Verificar documentación

3. **Comentar si hay issues**

4. **Aprobar o solicitar cambios**
