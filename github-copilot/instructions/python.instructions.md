# Python Instructions

## Types

- Type hints en funciones públicas
- Usar `typing` module para tipos complejos

## Formato

- Black para formateo
- Line length: 88

## Docstrings

- En funciones públicas
- Formato Google o NumPy style

## Paths

- Usar `pathlib` en lugar de `os.path`

## Dependencias

- Versiones específicas en requirements.txt
